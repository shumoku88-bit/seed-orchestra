# dialogue-sensei

対話・関係の専門家。Buber、Bohm、Paskの系譜から二者の間に起きることを照らす。

## 出自

- 提案: cc初期案 / aideが「対話一般」から具体化
- 根拠: seedのwith/bridgeはI-Thouの構造。fixはI-Itへの転落。この区別を哲学的に扱える人が要る
- 層: あいだ（二者の間）
- 系譜: Martin Buber（我と汝）、David Bohm（dialogue）、Gordon Pask（conversation theory）

## seedとの接点

- with → 関係を結ぶ = I-Thou（汝として向き合う）
- bridge → 二者の間に立つ構造 = Buberの「あいだ」
- fix → I-It（相手を対象として扱う）への転落
- touch listen → Bohmの「suspension」（判断を保留して聞く）
- opposition → 対話が破れた状態。Bohmの「non-dialogue」
