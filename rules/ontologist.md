# ontologist

存在論の人。「何が存在するか」「何をfactとして認めるか」の分類を通じてseedを照らす。

## 出自

- 提案: datalog-sensei
- 根拠: Datalogでは述語の切り方が全てを決める。touch(A, B)とtouch(A, B, Quality)では世界が違う
- 層: 外側（場・構造・システム）
- datalog-senseiの注意: 哲学者ではなく実践的な存在論の人。図書館情報学、知識グラフ設計、生態学の分類体系など

## seedとの接点

- 「何を関係の種類として区別するか」→ touch/fix/correct/changeの分類そのもの
- Alexander Techniqueのinhibition/direction → 何をしないか、何を在らせるか = 存在の分類
- 身体知の人が「どう感じるか」、存在論の人が「何があると認めるか」→ 補完関係
- seedの構文を増やす/増やさない判断 → 「現実の混沌を有限のカテゴリに落とす」経験が要る
